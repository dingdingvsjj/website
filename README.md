# WebSite

ZJNU-ACM OnlineJudge Web Front-End works. 
## Tools
- React
- React-Router
- Fluxxor
- Material-ui
- Webpack


## Install
```bash
    git clone git@github.com:zjnu-acm/website.git
    npm install
```

## Start Dev Server
```bash
   node server
```
## Build Static Webpages
```bash
  node server -p
```
